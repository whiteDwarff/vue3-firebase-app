#<q-btn></q-btn>

button 요소 생성

-----

### option

- type
  button의 타입 정의
  [
    submit : 제출 버튼,
  ]
- flat 
  background 제거

- round 
  border-radius가 50% 적용된 둥근 버튼으로 변경

- dense
  button의 간격을 좁게 (fit)

- icon=""
  button의 아이콘 모양 정의
  [
    close : X 모양 아이콘,

  ]


- v-close-popup
  q-dialog, menu를 닫을 때 사용
  function 사용 가능
